What is Bootstrap?

Bootstrap is a free front-end framework developer tool that contains libraries of compiled and minified CSS and JS source code designed to make website building easier while minimizing file size and data usage. The tool updates existing HTML and CSS features to make them more consistent, and contains additional built-in features that can help format and style a page, granting more flexibility to the developer. It is also designed so that websites are functional and responsive across different electronic devices, including mobile devices. Bootstrap is great for developers who need a website up and running as soon as possible.

How do you use Bootstrap?

Developers can download Bootstrap or use BootstrapCDN, the cached version, if Internet access is available. Note that BootstrapCDN will always updated to the latest version, while downloading the tool requires constant updating.

To use BoostrapCDN, you need to link it to your HTML page. You also need to insert required meta tags and Javascript plugins for the tool to work.

Your head tag should be as follows:
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js"></script>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.14.7/umd/popper.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/js/bootstrap.min.js"></script>
</head>

Bootstrap Typography

The Bootstrap default for text is different from the browser default for text. This default applies to all text inside HTML body.
font-family: "Helvetica Neue", Helvetica, Arial, sans-serif;
font-size: 14px;
line-height: 1.428;
color:#333;

You can use different tags for the different effects:
  • Use the mark tag to highlight text.
  • Use the code tag to highlight code.
  • Use the blockquote tag to quote text.
    • Four score and seven years ago our fathers brought forth, upon this continent, a new nation, conceived in liberty, and dedicated to the proposition that all men are created equal.
